EuroSAT Model Adaptation Study

Benchmark, fine-tune, and export lightweight vision models for aerial imagery.

This project benchmarks modern efficient backbones on EuroSAT, adapts the strongest models with supervised transfer learning, and packages the best practical candidate as an ONNX deployment artifact.

Project flow

  1. Validate the local EuroSAT cache and benchmark the corrected 10-class split.
  2. Fine-tune the strongest candidate models with a staged transfer recipe.
  3. Export the best practical model as FP32 and INT8 ONNX artifacts.

Methodology

Interpret the baseline correctly

The baseline resets the classifier head before evaluation, so low accuracy should be read as a transfer lower bound rather than a true zero-shot claim.

What changed in this project

From screening run to publishable pipeline

  • The dataset cache is checked before the run so incomplete EuroSAT copies are rejected early.
  • The fine-tuning stage uses a staged recipe instead of a single fixed loop.
  • The deployment stage shows only the artifacts that were actually generated and validated.
